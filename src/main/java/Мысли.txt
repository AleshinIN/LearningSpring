    Создать
Написать класс/метод, который будет помогать измерять время участка кода, с возможностью установления временных флагов
        в теле программы(флаг старта, промежуточный флаг, флаг завершения)


    Изучить
Maven
плюсы и минус IoC - инверсия управления(объекты создаются вне класса)
Dependency Injection - внедрение зависимостей, т.е. переменная передается в класс, а не создается в нем (в appCont можно создать объект,
    который принимает на вход другой объект из appCont)
